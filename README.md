Оценка ИЦБ ДОМ.РФ состоит из моделирования платежей по облигации с последующим расчетом на их основе ценовых метрик: НКД, грязной/чистой цены, Z- и G- спредов, доходности к погашению (YTM), дюрации Маколея, модифицированной дюрации, а также требуемой надбавки к Ключевой ставке (в случае облигаций с плавающей ставкой купона). В рамках данной методики актуализирован конвенциональный подход к оценке ИЦБ ДОМ.РФ, описанный в первой версии Ценовой конвенции для ипотечных ценных бумаг, с целью его приведения в соответствие текущим стандартам и практикам рынка, а также учета накопленного опыта моделирования. Обновление методики произведено по двум направлениям: методологическому и инфраструктурному

**Методологические обновления:**
1.	Использование CPR по S-кривой вместо 6-месячного среднего исторического CPR в качестве базового значения модельного CPR позволяет модели оперативно реагировать на изменения ставки рефинансирования ипотеки, а также решает проблему отсутствия исторического CPR в первые месяцы обращения выпуска
2.	Формализован алгоритм оценки S-кривой на данных отчетов сервисных агентов выпусков ИЦБ ДОМ.РФ
3.	В целях исправления недооценки суммы погашений по графику при моделировании денежного потока по пулу закладных введена скорректированная формула WAM (вместо классической формулы WAM)
4.	Посредством моделирования денежного потока по пулу закладных по месяцам, а не по купонным периодам, значительно уточнены модельные суммы погашений по графику и процентных поступлений
5.	Прописан алгоритм расчета ценовых метрик для выпусков ИЦБ ДОМ.РФ с плавающей ставкой купона (Ключевая ставка + фиксированная надбавка)
6.	Для выпусков ИЦБ ДОМ.РФ с переменной ставкой купона уточнен расчет ставки купона: применены более точные формулы для расчета расходов Ипотечного агента, а также сумм процентных поступлений за первый расчетный период, учтены недополученные процентные поступления при полном погашении кредитов
7.	Формализован расчет WAC и WAM по Реестру ипотечного покрытия на дату подписания решения о выпуске, задающий значения WAC и WAM до появления первого отчета сервисного агента по выпуску
8.	Введено использование при расчете данных отчетов для инвесторов на будущие даты купонных выплат (в случае их наличия), в которых указаны значения погашения номинала и выплаты купона по облигации
9.	Детализированы алгоритмы расчета ценовых параметров при всех возможных типах расчета (фиксация Z-спреда, фиксация G-спреда, фиксация грязной/чистой цены облигации, фиксация требуемой надбавки к Ключевой ставке, фиксация ставки купона при размещении)
10.	Добавлена возможность проводить расчет на любую дату оценки в прошлом с использованием только тех данных, которые были доступны на указанную дату оценки, что позволяет восстановить расчет на момент транзакции (например, в целях расчета ожидаемой доходности и спредов)
11.	Формализовано округление платежей до копеек в соответствии с эмиссионной документацией ИЦБ ДОМ.РФ

**Инфраструктурные обновления:**
1.	«Калькулятор ИЦБ ДОМ.РФ» — интерактивная онлайн-реализация методики на сайте Ипотечного агента ДОМ.РФ
2.	Реализация методики на языке Python опубликована в открытом доступе на GitHub
3.	Добавлена возможность проведения расчета запросом по API
4.	Алгоритмы методики детально и подробно описаны, что значительно упрощает ее программную реализацию на стороне участника рынка (в случае необходимости)
5.	Все данные, необходимые для расчета, классифицированы в рамках единого набора, доступного по запросу по API
6.	Каждый расчет по методике сопровождается Excel-файлом, по которому можно пошагово проследить работу всех формул (Excel-файл с расчетом доступен для скачивания на сайте Калькулятора)
7.	В целях удобства навигации между методикой, ее реализацией на Python и Excel-файлом все элементы расчета имеют свой порядковый номер и машинной обозначение
8.	В методику добавлены учебные материалы, объясняющие структуру амортизации ИЦБ ДОМ.РФ, а также применение G- и Z-спредов при оценке амортизируемых облигаций
